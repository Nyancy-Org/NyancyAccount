## 项目介绍

这是一个提供基于 OAuth2.0 标准协议的账号快捷登录服务平台，可以帮助开发者快速实现账号快捷登录功能。

- 图标及形象来自画师[甘城なつき](https://www.pixiv.net/users/3036679)笔下的猫羽雫

### 官方网站

[https://account.lazy.ink/](https://account.lazy.ink)
